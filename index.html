1. "日西じほ"とは

きっとこのページを見ている人は
"日西じほってなんや？どうでもいいわ！！！！ぶっ飛ばすぞ！！！！！！"
とか思ってらっしゃると思うので簡単に紹介しますね‼‼！


まず簡単に言うとすれば多機能無能Botです。
下記の機能が使えます。

スラッシュコマンド
- time [時刻(t),日付(d),日時(dt)] : 時刻を教えてくれます(取得元:ntp.nict.jp)
- ntp_time [アドレス] : NTPサーバーから時刻を引っ張ってきます。(なにも入力しないとntp.nict.jp)
- epochlink_time [アドレス] : EpochLinkサーバーから時刻を引っ張ってきます。(なにも入力しないとelp.emptybox.win)
- about : 自己紹介してくれます。
- sarashi_help : コマンド一覧を出力します。
- totsuzen_no_shi [好きな言葉] : 好きな言葉から"突然の死"を作ってくれます。
- twitter_fix [Twitterの投稿URL] : twitterのURLをOGP対応URLに直してくれます(使用アドレス:vxtwitter.com,fixvx.com)
- oishi_yummy : 美味しいヤミー❗️✨🤟😁👍✨⚡️感謝❗️🙌✨🙌✨🙌感謝❗️🙌✨etc.
- janken [グー,チョキ,パー] : 最新のコンピューター技術を応用して高精度な勝率を誇ります(?) 「グー」「チョキ」「パー」で入力してください。
- koizumi [辞任する] : 小泉構文。「〇〇していた」や「〇〇する」、「〇〇かもしれない」などを入力してね～
- frieren [いろいろ] : フリーレン構文。
- kyoiku_shikei [教育][死刑] : ビッグモーター副社長構文。教育教育死刑死刑━━━━━━━━━━━━━━━━
- weather [都市名] : 天気予報です！天気予報表示します。
- chikaiyo_getsuyo : どぅいどぅいどぅ～
- random: 乱数生成
- join: ボイスチャンネルに入ります
- leave: ボイスチャンネルから抜けます
- start_pomodoro: ポモドーロテクニック？トマトタイマー？25+5分のタイマーを始めます！(全員が落ちるかleaveするかstop_pomodoroするまで永遠に繰り返します)
- stop_pomodoro: ポモドーロテクニックをやめます！

アプリコマンド
- GGRKS : 発言をGoogle検索にぶち込みます(?)
- 突然の死 : 発言を突然の死させます(???)
- ピン留め/解除 : メッセージのピン留めができます！リアクションでピン留めはBotがオンラインになってるときに送信されたメッセージのみピン留めと解除ができますが、アプリコマンドのピン留め/解除はいつ送信されたメッセージでもピン留めと解除ができます

その他機能
- メッセージのピン留め : ピン留めしたいメッセージに:pushpin:(📌)をリアクションするとメッセージをピン留めできます！リアクションをすべて解除するとピン留めは解除されます！Botがオンラインになっているうちに送信されたメッセージのみ使用可能です！(Botの再起動やアップデートが後はそれ以前につけたリアクションのピンの操作をしてもピン留めに反映されません。アプリコマンドのピン留め/解除を使用してください)

ちなみに"日西じほ"の読みですが、「さらしじほ」です。晒←分解したら日西ですね！(??????)
